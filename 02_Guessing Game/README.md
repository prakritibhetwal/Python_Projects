# Number Guessing Game

Simple interactive number-guessing CLI game.

- **Main:** See [02_Guessing Game/main.py](02_Guessing%20Game/main.py#L1)

## Description
Pick a difficulty level and try to guess the randomly chosen number. The game tells you whether your guess is too low or too high and counts attempts.

## Levels
- **Easy:** 1–50  
- **Medium:** 1–100  
- **Hard:** 1–500

## Run
From the repository root:
```bash
python "02_Guessing Game/main.py"
