# Calculator (CLI)

A command-line calculator supporting addition, subtraction, multiplication,
division, modulus, and exponentiation.

## How to run
python main.py

## Features
- Input validation (rejects non-numeric input, blocks divide-by-zero, blocks invalid exponent cases)
- Loop until user chooses to stop

## What I learned
- Functions, match-case, dictionaries, while loops, error handling with try/except
